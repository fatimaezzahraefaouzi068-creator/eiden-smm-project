# SocialPulse AI - Social Media Management Dashboard

## Stack technique
- **Backend**: Flask, PostgreSQL, Redis, Celery, Pandas
- **Frontend**: React, Tailwind CSS, Recharts, React Query
- **Containerisation**: Docker, Docker Compose

## Installation

### Prérequis
- Docker Desktop
- Node.js 18+

### Lancer le projet

```bash
# Cloner le projet
git clone https://github.com/fatimaezzahrafeaouz068-creator/eiden-smm-project.git
cd eiden-smm-project

# Lancer le backend
docker-compose up -d

# Lancer le frontend
cd frontend
npm install
npm run dev
